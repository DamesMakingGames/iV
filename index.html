iV

A system for making synchronous, hackable, interactive narrative games.

Authors and Contributors

Currently in development by @jennie, @cecilycarver and @dann for Dames Making Games.

Built with Daimio: https://github.com/dxnn/daimio

Original concept and development by Alex Leitch for Hannah Epstein's synchronous multi-screen/device installation game PsXXYborg.

Support or Contact

Need help? Tweet at @jennie

Installation

  1. Install Node.js
  2. $ git clone git@github.com:bentobx/iV.git
  3. $ npm install mongodb
  4. $ node daimio_sp_app.js
  5. go to server.foo:8888

Need help? Tweet at @jennie

History

Concept and development by Alex Leitch for Hannah Epstein's synchronous multi-screen/device installation game PsXXYborg. Additional development support by Dann Toliver (@dxnn).
